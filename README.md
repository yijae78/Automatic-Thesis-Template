# 오이코스 논문 템플릿 자동 채우기

학교 공식 논문 워드 템플릿에 내용을 채워서 DOCX로 내보내는 프로그램입니다.

## 실행 방법

1. **한 번만**: 터미널에서 `npm install` 실행 (express, multer 등 설치)
2. **서버 실행**: `run_server.bat` 더블클릭 또는 터미널에서 `npm start`
3. 브라우저에서 **http://localhost:3750** 접속

## 사용법

- **왼쪽(내용 입력)**: 제목, 이름, 날짜, 초록, 헌정, 제1~7장 등 입력. 수정은 여기서만 가능.
- **오른쪽(미리보기)**: 입력 내용이 읽기 전용으로 표시. 일반 모니터에서 왼쪽·오른쪽 50:50 비율.
- **채팅창**: `제목: ...`, `이름: ...`, `날짜: ...` 형식으로 입력해도 반영됨.
- **입력 내용 저장**: 현재 입력을 브라우저(localStorage)에 저장. 같은 PC·같은 브라우저에서만 유지.
- **저장된 내용 불러오기**: 저장된 내용을 폼에 다시 채움. 페이지를 열면 저장된 내용이 있으면 자동으로 불러옴.
- **DOCX에서 불러오기**: 이미 작성한(또는 작성 중인) 워드 파일을 선택하면, 해당 내용이 왼쪽 폼에 채워짐. 이 프로그램으로 만든 DOCX 또는 동일 오이코스 템플릿 구조 문서에 적합.
- **DOCX 다운로드**: 채운 내용으로 워드 파일 생성 후 다운로드

## CLI로만 채우기

- `data.json`을 수정한 뒤 터미널에서 `node fill_docx.js` 실행 → `filled.docx` 생성
